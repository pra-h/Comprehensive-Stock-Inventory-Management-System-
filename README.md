# Stock Inventory Management System

A comprehensive stock inventory management system developed using **Microsoft Excel and VBA**, designed to automate inventory tracking and stock updates for       small-scale organizations.

## Key Features
- Automated stock entry and updates using VBA macros  
- Real-time inventory tracking and validation  
- Automatic calculation of stock levels  
- User-friendly Excel-based interface  

## Technology Stack
- Microsoft Excel (Macro-Enabled)  
- VBA (Visual Basic for Applications)
